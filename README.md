# OLA Ride Analytics Dashboard | Power BI

## Project Overview
An interactive 5-page Power BI dashboard analyzing 103,024 OLA ride bookings 
worth ₹35.08M, covering revenue, cancellations, vehicle performance, and ratings.

## Dashboard Pages
- **Overall** – Total bookings, booking value, ride volume over time
- **Vehicle Type** – Performance comparison across 7 vehicle types
- **Revenue** – Payment method breakdown (Cash, UPI, Credit Card, Debit Card)
- **Cancellation** – 28.08% cancellation rate analysis
- **Ratings** – Driver and customer ratings by vehicle type

## Key Insights
- Cash is the most used payment method, followed by UPI
- Cancellation rate stands at 28.08% with 28,933 cancelled bookings
- Average driver and customer ratings hover around 4.0 across all vehicle types

## Tools Used
- Power BI Desktop
- DAX for calculated measures
- Data modeling and Star Schema
